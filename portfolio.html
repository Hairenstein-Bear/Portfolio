Paul's Project Portfolio

Analog Clock

I created this analog clock using a combination of HTML, CSS, and Javascript in VsCode. I wanted to display a clock on the main page of this site, however a digital clock seemed too easy. The decision paid off, because making this application taught me a lot about integrating javascript with css.

Sample Randomizer

I created this for a client who needed to conduct random tests on prepared medication. The application is being used to reduce human bias in quality assurance testing for pharmaceutical compounding. It is built with mostly CSS, using Javascript to handle the randomization, the application was coded in VsCode.

Frog Chase Game

I made this as a birthday present for my nephew. I had a blast making it and he loved to play it. I used Javascript to draw the canvas as well as the player character,food, and track score. I used an old snake game I had made in college, and heavily modified it in VsCode to make it more relevant to my nephew.

Wolveshead Pizza and Wings

I have been administering my employers WordPress website for the last two years, and recently decided to remake it from scratch to allow for more flexibility in page design. I used HTML, CSS, PHP and Javascript as well as incorporated jquery frameworks into the construction of the site.
